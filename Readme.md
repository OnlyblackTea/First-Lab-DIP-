# 实验目的
1. 设计或运用某一种或多种算法对给出的3幅图像(test1_1.jpg, test1_2.jpg, test1_3.jpg)进行增强处理。
2. 不局限于第三章所介绍算法，可利用其他自已课外学习的算法。
3. 可以自行扩充其他图片验证算法效果。


# Installation
1. **确保你已经安装了 Poetry**。
2. 克隆项目仓库:
```bash
git clone git@github.com:OnlyblackTea/First-Lab-DIP-.git
```
3. 安装项目依赖
```bash
poetry install
```
4. 激活虚拟环境（可选）：
```bash
poetry shell
```
